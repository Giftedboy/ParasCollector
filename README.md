# 可能存在的BUG
* 中文参数名，未做特殊处理，想来应该不会有这么变态的开发
* 数据包过多时（几万+），self._paras参数会占用较多内存，且存在不稳定的情况，可能最后没能生成文件，所以建议是在黑名单内排除不需要收集的域名以及path，至于为什么不做可视化的输入黑名单呢？别问，问就是不会😭😭😭
